# Architecture Documentation

This folder contains all architecture-related documentation including:
- System architecture diagrams
- Design documents
- Architecture analysis
- RAG system architecture
- System design documents (SDD)
- Product requirements (PRD)

## Contents

- **ARCHITECTURE_SUMMARY.md** - High-level architecture summary
- **DATABASE_SCHEMA_RECIPES.md** - Database schema for recipes
- **diagrams.md** - Mermaid diagrams for architecture visualization
- **DUPLICATE_RECIPE_DETECTION.md** - Duplicate recipe detection system documentation
- **DUPLICATE_RECIPE_DETECTION_UML.md** - UML diagrams for duplicate detection
- **DUPLICATE_RECIPE_DETECTION_FLOW.json** - JSON flow definition for duplicate detection
- **n8n-config.md** - n8n workflow configuration
- **PRD.md** - Product Requirements Document
- **RAG.md** - RAG system architecture
- **SDD.md** - System Design Document
- **SECURITY.md** - Security documentation
